Polecam zapoznanie się z tymi materiałami. Znajdziemy tam natywny system do pisania makr, zbliżony do VB z MS Office. Możliwości są całkiem spore. Dzisiaj napisałem krótkie makro, którego zadaniem jest wstawienie nowego slajdu do prezentacji i umieszczenie na nim prostokąta. Potem makro automatycznie uruchamia pokaz slajdów.

Sub Jedziemy
	
	Dim Doc As Object
	Dim Page As Object
	Dim Square As Object
	Dim Point As New com.sun.star.awt.Point
	Dim Size As New com.sun.star.awt.Size
	 
	Doc = ThisComponent
        ' add new page at the end
	Page = Doc.DrawPages.insertNewByIndex(Doc.getDrawPages.getCount())

	Point.x = 3000
	Point.y = 3000
	Size.Width = 3000
	Size.Height = 3000
	' create square drawing element
	Square = Doc.createInstance("com.sun.star.drawing.RectangleShape")
	Square.Size = Size
	Square.Position = Point
	Square.FillColor = RGB(255,128,128) 
	Page.add(Square)
	
	' odpalamy prezentacje
	Doc.Presentation.Start

End Sub

Takie makro można wrzucić do Narzędzia->Makra->Zarządzaj makrami->Makro... Tam w kategorii "Moje makra" tworzymy nowe makro i wklejamy ten kod. Możemy również potem kliknąć przycisk "Przypisz" i przypisać nasze makro do czynności "Otwórz dokument" - dzięki temu makro otworzy się zaraz po otwarciu dokumentu.
Ostatnio zmieniany: 2 lata, 2 mies. temu przez bartnikj.
