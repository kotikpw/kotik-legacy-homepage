Odp: HTML5 + WebGL
nerull7 napisał:
W javascriptcie jest obiekt XMLHttpRequest który służy do pobierania stron przez js, dzięki niemu można aktywnie modyfikować stronkę podczas jej użytkowania (np. podpowiedzi wyszukiwania na stronie googla). W HTLM5 jest WebSocket który jest tzw. tcp dla html.
Bardzo sympatyczne. Zrobione jest na obiekcie <canvas> dodawanym po załadowaniu strony. Sporo skryptowania ale możliwości są właściwie nieograniczone. Tylko czy nie uciekamy wtedy za bardzo od WebGL (które i tak korzysta z obiektu canvas)?

A może WebGL nie jest nam potrzebny, a wystarczy jedynie HTML5? Canvas z HTML5 obsługiwany jest prawie przez każdą przeglądarkę (z najpopularniejszych chyba tylko Opera tego jeszcze nie ma). WebGL chodzi właściwie tylko na betach FF, Safari i Chromium i odpaleniu specjalnego polecenia lub grzebaniu w ustawieniach.

Tak sobie myślę, że etapem do którego powinno się zmierzać jest mocno okrojona dystrybucja, najlepiej w wersji live odpalanej np. z karty CF (podłączonej jako dysk twardy), zabezpieczonej przed zapisem, z całym naszym SPT... tak żeby każdy mógł sobie taki kiosk postawić.

Zostawiam do przemyśleń.
Pozdrawiam,
Jan Bartnik.

OpenOffice.org BASIC - materia ...
Polecam zapoznanie się z tymi materiałami. Znajdziemy tam natywny system do pisania makr, zbliżony do VB z MS Office. Możliwości są całkiem spore. Dzisiaj napisałem krótkie makro, którego zadaniem jest wstawienie nowego slajdu do prezentacji i umieszczenie na nim prostokąta. Potem makro automatycznie uruchamia pokaz slajdów.

Sub Jedziemy
	
	Dim Doc As Object
	Dim Page As Object
	Dim Square As Object
	Dim Point As New com.sun.star.awt.Point
	Dim Size As New com.sun.star.awt.Size
	 
	Doc = ThisComponent
        ' add new page at the end
	Page = Doc.DrawPages.insertNewByIndex(Doc.getDrawPages.getCount())

	Point.x = 3000
	Point.y = 3000
	Size.Width = 3000
	Size.Height = 3000
	' create square drawing element
	Square = Doc.createInstance("com.sun.star.drawing.RectangleShape")
	Square.Size = Size
	Square.Position = Point
	Square.FillColor = RGB(255,128,128) 
	Page.add(Square)
	
	' odpalamy prezentacje
	Doc.Presentation.Start

End Sub

Takie makro można wrzucić do Narzędzia->Makra->Zarządzaj makrami->Makro... Tam w kategorii "Moje makra" tworzymy nowe makro i wklejamy ten kod. Możemy również potem kliknąć przycisk "Przypisz" i przypisać nasze makro do czynności "Otwórz dokument" - dzięki temu makro otworzy się zaraz po otwarciu dokumentu.
